# HR Analytics Dashboard (Excel Project)

This project is a complete **HR Analytics Dashboard** built in Microsoft Excel using the IBM HR Employee Attrition Dataset.  
It includes KPI cards, pivot tables, and interactive charts to provide insights into employee headcount, salary distribution, and attrition trends.

---

## üìä **Project Overview**

The purpose of this project is to analyze HR data and present it in a clean, professional dashboard.  
The dashboard helps HR teams quickly identify:

- Workforce distribution by department  
- Salary patterns and compensation differences  
- Attrition count and attrition rate  
- Key HR metrics such as total employees, average salary, and overall attrition  

This project demonstrates practical skills in:
- HR Analytics  
- Excel Data Cleaning  
- Pivot Tables  
- Pivot Charts  
- Dashboard Design  
- KPI Visualization
